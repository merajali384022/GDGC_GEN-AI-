### Cricket Player Persona Generator Using Groq API

This project generates realistic cricket player personas using Prompt Engineering techniques and the Groq API.

### Features

- Generates 3 realistic cricket personas
- Structured JSON output
- Uses Groq Llama3 model
- Applies Prompt Engineering techniques

### Technologies Used

- Python
- Groq API
- Prompt Engineering
- JSON

### Prompt Engineering Techniques Used

- Role Prompting
- Structured Prompting
- Few-shot Prompting
- Temperature Tuning
